## Troubleshooting

**Q: I don't see the files in GitHub Desktop's Changes tab**
A: 
1. Close GitHub Desktop completely
2. Go to repository folder
3. Delete the `.git` folder
4. Reopen GitHub Desktop
5. File → Add Local Repository → Select folder
6. Check Changes tab again

**Q: Files are showing but I can't select them**
A:
1. Make sure you're on the "Changes" tab (not History)
2. Look for checkboxes next to each filename
3. Click checkbox to select individual files
4. Or look for "Select All" option if available

**Q: Commit button is grayed out**
A:
1. You must SELECT at least one file first
2. You must enter a commit message
3. Try refreshing: Repository → Refresh

**Q: Push failed**

1. Check your GitHub login in GitHub Desktop
2. Try logging out and back in
